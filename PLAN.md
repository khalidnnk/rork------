# إضافة مدن عربية لقائمة الاختيار اليدوي

## الخطة

### الميزة
- توسيع قائمة المدن لتشمل مدن عربية من عدة دول (السعودية، مصر، الإمارات، الكويت، قطر، البحرين، عُمان، الأردن، العراق، لبنان، سوريا، فلسطين، ليبيا، تونس، الجزائر، المغرب، السودان، اليمن)
- تصنيف المدن حسب الدولة لسهولة البحث
- إمكانية البحث في المدن بالاسم العربي أو الإنجليزي

### المدن المضافة (أمثلة)
- **السعودية**: المدن الحالية + الطائف، ينبع، الأحساء، الجبيل
- **مصر**: القاهرة، الإسكندرية، الجيزة، أسوان
- **الإمارات**: دبي، أبوظبي، الشارقة، العين
- **الكويت**: مدينة الكويت، الأحمدي، حولي
- **قطر**: الدوحة، الريان
- **البحرين**: المنامة، المحرق
- **عُمان**: مسقط، صلالة
- **الأردن**: عمّان، إربد، الزرقاء
- **العراق**: بغداد، أربيل، البصرة، النجف، كربلاء
- **فلسطين**: القدس، غزة، رام الله، الخليل
- **لبنان**: بيروت، طرابلس
- **سوريا**: دمشق، حلب
- **اليمن**: صنعاء، عدن
- **السودان**: الخرطوم
- **ليبيا**: طرابلس، بنغازي
- **تونس**: تونس، صفاقس
- **الجزائر**: الجزائر، وهران
- **المغرب**: الرباط، الدار البيضاء، مراكش

### التصميم
- في صفحة الإعدادات ضمن قسم "الموقع"، يظهر زر **"اختيار المدينة يدوياً"**
- عند الضغط يفتح شاشة منبثقة (Modal) تعرض المدن مصنفة حسب الدولة
- شريط بحث في أعلى الشاشة للبحث بالاسم
- كل دولة تظهر كعنوان فرعي وتحتها مدنها
- عند اختيار مدينة يتم تحديث الموقع تلقائياً وإغلاق الشاشة
- التصميم يتبع نفس أسلوب التطبيق الحالي (ألوان داكنة مع ذهبي)
